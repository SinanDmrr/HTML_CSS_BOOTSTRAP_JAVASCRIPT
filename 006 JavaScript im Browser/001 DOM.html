Das Document Objekt Model (DOM)

Das DOM ist die Schnittstelle zwischen HTML und dynamsichem JavaScript.
Alle Elemente werden zu Objekten, die dynmaisch aufgerufen, verändert, hinzugefügt
und gelöscht werden können.


Wie es funktiniert:

  1. Website liegt dem Browser als bloßer Text vor.
  2. Während des empfangens des Codes, verarbeitet ihn der Parser des Browsers Stück für Stück
  3. Der Parser überführt den Code in eine Objektstruktur. Diese wird im RAM gehalten.
    Die Struktur besteht aus verschaltelten Knoten.
    1. Elementknoten
    2. Attributknoten
    3. Textknoten
    Diese ergibt eine Baumstruktur.
  4. Der Browser nutzt für die Weiterverarbeitung nun nicht mehr den HTML Code sondern genau diese
    Objektstruktur aus dem RAM    . CSS und JS beziehn sich ebenfalls auf diese Baumstruktur.

Farben in CSS

Hexadezimal: Rot-Wert, Grün-Wert, Blau-Wert

Info: Werte von 0-9 und dann A-F (Insagesamt 15)

Rot: 255 | 0 | 0

Grün: 0 | 255 | 0

Blau: 0 | 0 | 255

RGBA: Rot-Wert, Grün-Wert, Blau-Wert, Alpha-Wert

Info: Werte von 0 bis 255 & Deckkraft/Transparenz 0% bis 100%

Rot: 255 | 0 | 0 | 25%

Grün: 0 | 255 | 0 | 25%

Blau: 0 | 0 | 255 | 25%

HEX: Rot-Wert, Grün-Wert, Blau-Wert

Info: Werte von 0 bis 255 & Deckkraft/Transparenz 0% bis 100%

Rot:# FF | 00 | 00

Grün:# 00 | FF | 00

Blau:# 00 | 00 | FF

HSL: Hue (Farbton), Saturation (Sättigung), Lightness (Helligkeit)

Info: Hue-Werte von 0 bis 360 und die anderen 0% bis 100%

Rot: 250| 100% | 50%

Grün: 250| 100% | 50%

Blau: 250| 100% | 50%

CSS-Code Auslagern

Um den CSS Style Teil auszulagern müssen wir eine StyleSheet Datei erzeugen.

1. Neue Datei -> style.css
2. Alles was zwischen den "style-Tag" im HTML Dokument steht wird raus kopiert
3. Der Kopierte Inhalt wird nun der Datei "style.css" hinzugefügt, beachte dabei das zwischen den Style Tags der inhalt eingefügt wird nicht das Style mit kopieren!
4. Die CSS Datei ins HTML Verlinken im Head-Bereich:
Beispiel: link rel="stylesheet" type="text/css" href="style.css"

Nun schaut die Seite wie zuvor aus mit dem Style nur das wir es ausgelagert haben und somit die
HTML Datei und die CSS Datei separiert und Sauberer nun sind.

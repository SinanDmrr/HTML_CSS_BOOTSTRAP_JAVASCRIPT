Float & Clear Konzept
Beim Float wird das Element zum Inline Element und nimmt nur noch den benötigten Platz ein
Desweiteren Schwebt es, sodass die anderen Elemente hoch rutschen, wodurch die hinter dem
Float Element verschwinden also aufpassen! Clear hebt das Float auf also Float:left kann mit Clear:left aufgehoben werden
